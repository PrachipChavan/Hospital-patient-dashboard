📋 Dataset Overview
The dataset (hospital_patients_data.csv) contains 1,200 rows and 17 columns of realistic hospital patient data:

Column	Type	Description
PatientID	Text	Unique patient ID (e.g. PAT-0001)
PatientName	Text	Full name of the patient
Age	Number	Age of the patient
Gender	Text	Male / Female
AdmissionDate	Date	Date of hospital admission
DischargeDate	Date	Date of discharge (blank if still admitted)
LengthOfStay	Number	Number of days in hospital
Status	Text	Admitted or Discharged
AdmissionType	Text	Emergency, Elective, or Urgent
Department	Text	Medical department (e.g. Cardiology, Oncology)
TreatingPhysician	Text	Name of the assigned doctor
Diagnosis	Text	Medical diagnosis
RoomNumber	Text	Hospital room number
BillingAmount	Decimal	Total billed amount in USD
InsuranceProvider	Text	Insurance carrier or Self-Pay
SatisfactionRating	Number	Patient rating from 1 (Poor) to 5 (Excellent)
ReadmissionRisk	Text	Low, Medium, or High

🧮 DAX Measures
The following DAX measures power the KPI cards and are used throughout the dashboard:

dax

Total Admissions = COUNTROWS('hospital_patients_data')
Active Patients = CALCULATE(COUNTROWS('hospital_patients_data'), 'hospital_patients_data'[Status] = "Admitted")
Avg Length of Stay = AVERAGE('hospital_patients_data'[LengthOfStay])
Total Billed = SUM('hospital_patients_data'[BillingAmount])
Avg Satisfaction = AVERAGE('hospital_patients_data'[SatisfactionRating])
High Readmission Risk % = 
DIVIDE(
    CALCULATE(
        COUNTROWS('hospital_patients_data'), 
        'hospital_patients_data'[ReadmissionRisk] = "High",
        'hospital_patients_data'[Status] = "Discharged"
    ), 
    CALCULATE(
        COUNTROWS('hospital_patients_data'),
        'hospital_patients_data'[Status] = "Discharged"
    ),
    0
)
